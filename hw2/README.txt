README

The files included in the zip file are:
hw2.R: script file for homework 2
hw2_funcs.R: functions needed to run homework 2.
hw2_report.pdf: The homework 2 report.
adult.data.txt: text file containing the combined test and training data from https://archive.ics.uci.edu/ml/datasets/Adult.
group.txt: list of group members

Instructions on Running Code:
Run the whole script for hw2_funcs.R, and then run the whole script for hw2.R. Make sure that the package caret is installed.
